Plataforma de Aprendizagem Adaptativa

Com uso de Learning Analytics e Machine Learning

Este projeto visa personalizar o ensino com base nas necessidades de cada aluno, utilizando inteligência artificial para melhorar o aprendizado.

Funcionalidades

Identificação de pontos fortes e fracos dos alunos, sugestões de conteúdo e relatórios de desempenho.

Tecnologias

Bootstrap, TypeScript, HTML, Machine Learning, Learning Analytics.

Diferenciais

Plataforma inteligente, personalizada e baseada em dados reais de desempenho.
